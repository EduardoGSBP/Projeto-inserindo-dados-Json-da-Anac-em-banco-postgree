# 📊 ETL com JSON da ANAC para PostgreSQL

Projeto simples de ETL (Extração, Transformação e Carga) utilizando Python, onde dados em formato JSON são tratados e inseridos em um banco PostgreSQL.

---

## 🚀 Tecnologias

* Python
* Pandas
* PostgreSQL
* Psycopg2

---

## ⚙️ Etapas do projeto

1. Leitura de arquivo JSON como DataFrame
2. Seleção das colunas relevantes
3. Padronização dos nomes (remoção de acentos)
4. Criação do banco e tabela no PostgreSQL
5. Configuração da conexão com o banco
6. Inserção dos dados na tabela
7. Limpeza da tabela antes da carga (mantendo dados atualizados)
8. Refatoração do código
9. Conversão de `.ipynb` para `.py`

---
